## 1. 전화
전화번호를 입력하고 통화 버튼을 누르면 통화음과 함께 통화화면으로 전환<br/><br/>
![전화](https://github.com/tpdus001129/test/assets/113432040/2abfa6e9-1f73-471a-b8a9-5f12992cbca4)

## 2. 날씨

[openweathermap api 이용](https://openweathermap.org/)<br/><br/>
